# Power BI Internship Project  
## Human Resources Analytics and Educational Performance Dashboard

This repository contains two interactive Power BI dashboards developed as part of the internship program requirements. The projects focus on analytical reporting, KPI monitoring, and data-driven decision support.

---

# Project 1: Human Resources Analytics Dashboard

## Objective
To analyze and optimize HR processes, employee performance, and workforce management using interactive Power BI visualizations.

## Dashboard Overview
The HR Analytics Dashboard provides insights into employee attrition, workforce distribution, salary analysis, and performance metrics. It enables management to identify turnover patterns, departmental performance gaps, and employee engagement factors.

## Key Metrics

- Total Employees
- Attrition Count
- Attrition Rate (%)
- Average Monthly Income
- Average Years at Company

## Key Analysis Performed

- Attrition analysis by department
- Attrition trend by years at company
- Work-life balance distribution analysis
- Department-wise performance evaluation
- Salary insights and income distribution
- Employee segmentation by job role

## Filters Implemented

- Department
- Gender
- Marital Status

## Business Value

- Identifies high attrition departments
- Supports workforce planning decisions
- Assists HR in improving employee retention strategies
- Enables performance-based organizational insights

# Project 2: Educational Performance and Resource Allocation Dashboard

## Objective
To evaluate student academic performance and support effective resource allocation in educational institutions through data-driven analysis.

## Dashboard Overview
The Educational Dashboard provides insights into student outcomes, dropout trends, academic performance across semesters, and course-wise comparisons. It helps institutions identify performance gaps and improve strategic decision-making.

## Key Metrics

- Total Students
- Dropout Percentage
- Graduation Percentage
- Average 1st Semester Grade
- Average 2nd Semester Grade

## Key Analysis Performed

- Student outcome distribution (Graduate vs Dropout vs Enrolled)
- Course-wise academic performance comparison
- Semester-wise performance trends
- Course-wise dropout and graduation analysis
- Scholarship impact analysis
- Tuition vs dropout correlation analysis

## Filters Implemented

- Course
- Gender
- Target Category (Graduate/Dropout/Enrolled)

## Business Value

- Identifies high-risk courses with higher dropout rates
- Supports academic performance monitoring
- Assists in scholarship impact evaluation
- Enables data-driven educational planning
